# fairbnb

una copia spudorata di airbnb scritta in java

## Prima di partire

Crea un file `.env` con le seguenti variabili da customizzare:

```
DB_HOST=localhost
DB_NAME=fairbnb
DB_USER=root
DB_PASS=root

ADMIN_EMAIL=davide.dicriscito@gmail.com
ADMIN_PASS=12345
ADMIN_FNAME=Davide
ADMIN_LNAME=Di Criscito
```
